CSS Inline

Inline-CSS wird direkt im HTML-Tag über das style-Attribut definiert, um spezifische Elemente schnell und lokal zu formatieren. Es überschreibt externe Stylesheets aufgrund seiner hohen Spezifität, führt jedoch bei häufiger Nutzung zu unübersichtlichem Code, erschwerter Wartung und längeren Ladezeiten, weshalb es primär für Tests oder kleine Anpassungen geeignet ist. Erwird direkt in einem Tag verwendent
[-p style="color: blue"-]

CSS Boxmodell


Die Komponenten des Boxmodells (von innen nach außen)
Content (Inhalt): Der eigentliche Inhalt, z. B. Text oder Bilder. Kann mit width: und height: angepasst werden über den [-style-] tag

Padding (Innenabstand): Transparenter Bereich direkt um den Inhalt. [Veränderbar über style="padding-top: 10px;"]

Border (Rahmen): Eine Linie, die Padding und Inhalt umgibt. [Veränderbar über style="border-top: 10px;"]

Margin (Außenabstand): Transparenter Bereich außerhalb des Rahmens, der Abstand zu anderen Elementen schafft. [Veränderbar über style="margin-top: 10px;"]

CSS Flexbox

CSS Flexbox Poster

Es wird der display: Flex; verwendent flex und grid wird im Hauptcontainer [parent] angewendet!

flex-direction: [Flex-Richtung] ist eine CSS-Eigenschaft im Flexbox-Modell, die festlegt, ob Flex-Elemente in einer Zeile (row) oder Spalte (column) angeordnet werden. Sie definiert die Ausrichtung der Hauptachse, wobei row (Standard) die Elemente horizontal und column vertikal anordnet. Zudem ermöglicht sie durch reverse-Werte die Umkehrung der Anordnung.

gap: XXpx; ist die Lücke zwischen den Reihen und den Spalten in Flexbox-, Grid- und Mehrspalten-Layouts.

justify-content: center; [Inhalt ausrichten] Sie richtet Flex-Items aus (z.B. zentriert, verteilt) und beeinflusst deren Position. Es ist entscheidend für responsives Design, da es sich an Bildschirmgrößen anpasst.

align-items: center; [Elemente ausrichten] ist eine CSS-Eigenschaft in Flexbox- und Grid-Layouts, die Flex-Items auf der Querachse (meist vertikal) ausrichtet. Sie wird auf den Container angewendet, um Kindelemente zu zentrieren (center), am Anfang/Ende zu platzieren (flex-start/flex-end) oder zu strecken (stretch, Standardwert). Sie zentriert Elemente vertikal, wenn flex-direction auf row steht.

CSS Pseudo Class

CSS-Pseudoklassen sind Schlüsselwörter, die mit einem Doppelpunkt (:) beginnen und an Selektoren angehängt werden, um HTML-Elemente basierend auf ihrem Zustand oder ihrer Position im Dokumentbaum zu stylen. Sie ermöglichen dynamische Stile, ohne Klassen im HTML hinzufügen zu müssen

:hover Stilt ein Element, wenn der Mauszeiger darüber schwebt.

:active Stilt ein Element, während es aktiviert wird (z. B. beim Anklicken).

:focus Stilt ein Element, wenn es den Fokus hat (z. B. durch Tab-Taste oder Klick).

:focus-visible Zeigt einen Fokusrahmen nur dann an, wenn er für den Nutzer sichtbar sein sollte (besser für Barrierefreiheit).

:focus-within Stilt ein Element, wenn es selbst oder eines seiner Kinder den Fokus hat.

CSS Spezifität

CSS-Spezifität ist der Algorithmus, mit dem Browser bestimmen, welche CSS-Regel auf ein Element angewendet wird, wenn mehrere konkurrierende Regeln existieren. Sie gewichtet Selektoren (ID > Klasse > Element) und entscheidet, welcher Stil Vorrang hat. Höhere Spezifität gewinnt, wobei Inline-Styles am stärksten sind.

Hierarchie: Die Gewichtung folgt einer Struktur: Inline-Styles (1,0,0,0) > ID-Selektoren (0,1,0,0) > Klassen/Attribute/Pseudoklassen (0,0,1,0) > Elemente/Pseudoelemente (0,0,0,1).

!important: Diese Regel überschreibt alle anderen Spezifitäten, sollte aber sparsam eingesetzt werden.
